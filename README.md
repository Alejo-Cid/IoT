# Trabajo Práctico: Servidor Web asincrónico con ESP32

El trabajo fue realizado usando platformIO con una esp32-cam (por que es la que tengo), en la interfaz se pueden ver los valores de temperatura y humedad, y apagar y prender el led. Se utiliza SPIFFS para el sistema de archivos (htl,css y javascript), y se usa WiFiManager para levantar un portal cautivo para configurar las credenciales. Tambien intente usar un sensor DHT22 que tenía pero haciendo pruebas de me di cuenta de que estaba quemado, por lo tanto termine simulando los valores que se actualizan cada 2s.
